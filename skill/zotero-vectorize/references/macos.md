# macOS notes

## Default Zotero data directory

Usually:

```text
/Users/<UserName>/Zotero
```

Important contents:

- `zotero.sqlite`
- `storage/`

## Confirm the real path

In Zotero:

- `Zotero`
- `Settings`
- `Advanced`
- `Show Data Directory`

## Common macOS pitfalls

- SQLite lock errors when Zotero is open
- LibreSSL/OpenSSL warnings from Python packages
- First-time model download timeouts
